# Reisen Buchungs App

Willkommen zur Reisen Buchungs App! Diese Anwendung hilft Ihnen bei der Planung und Buchung Ihrer Reisen.

## Funktionen
- Suche und buchen Sie Flüge und Unterkünfte.
- Erstellen Sie persönliche Reisepläne.
- Vergleichen Sie Preise und Bewertungen von Hotels.
- Teilen Sie Ihre Reiseerlebnisse mit anderen Nutzern.

## Technologien
- Frontend: React, Bootstrap
- Backend: Node.js, Express
- Datenbank: MongoDB
- Authentifizierung: JWT (JSON Web Tokens)

## Installation
1. Klonen Sie das Repository:
   `git clone https://github.com/IhrBenutzername/reisen-buchungs-app`
2. Installieren Sie die Abhängigkeiten:
   `npm install`
3. Starten Sie den Server:
   `npm start`
